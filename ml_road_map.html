PHASE 01 ⏱ 6–8 Weeks
Math & Programming Foundations
Everything ML is built on. Skip this and you'll hit a wall. Master it and every algorithm will make sense intuitively.
📐 Linear Algebra ▼
  • Vectors, matrices, tensors — operations and intuition
  • Matrix multiplication, transpose, inverse
  • Dot product, cross product, projections
  • Eigenvalues & eigenvectors (crucial for PCA)
  • SVD — Singular Value Decomposition
  • Norms (L1, L2) and their role in regularization
Resources
📺 3Blue1Brown — Essence of Linear Algebra (YouTube) 🎓 MIT 18.06 Linear Algebra (OCW) 📘 Mathematics for ML — Deisenroth (free PDF)
📉 Calculus & Optimization ▼
  • Derivatives, partial derivatives, chain rule
  • Gradient — the direction of steepest ascent/descent
  • Gradient Descent and its variants (SGD, Adam, RMSProp)
  • Backpropagation — the engine of deep learning
  • Convexity and loss landscapes
Resources
📺 3Blue1Brown — Essence of Calculus (YouTube) 🎓 Khan Academy — Multivariable Calculus
🎲 Probability & Statistics ▼
  • Probability basics — distributions, expectation, variance
  • Bayes' theorem (foundation of Bayesian ML)
  • Normal, Binomial, Poisson distributions
  • Hypothesis testing, p-values, confidence intervals
  • MLE — Maximum Likelihood Estimation
  • Information theory: entropy, KL divergence
Resources
📺 StatQuest with Josh Starmer (YouTube — essential!) 📘 Think Stats — Allen Downey (free online)
🐍 Python for ML ▼
  • Python fundamentals — loops, functions, OOP, list comprehensions
  • NumPy — arrays, broadcasting, vectorized operations
  • Pandas — dataframes, groupby, merging, cleaning
  • Matplotlib & Seaborn — visualizing data
  • Jupyter Notebooks — your daily workspace
Resources
📺 Corey Schafer Python (YouTube) 🎓 CS50P — Python (Harvard, free) ⚡ Kaggle Learn — Python + Pandas (free, hands-on) 🔧 Google Colab (free GPU)
⚡ Phase 1 Projects
  • EASY Implement gradient descent from scratch in NumPy
  • EASY Exploratory Data Analysis on a real dataset (Titanic, Iris)
  • MID Build a statistics dashboard with Matplotlib on COVID / agriculture data
PHASE 02 ⏱ 8–10 Weeks
Classical Machine Learning
The bread and butter. These algorithms solve 70% of real-world problems. Master them deeply — don't just call .fit().
📊 Supervised Learning — Regression ▼
  • Linear Regression — OLS, cost function, closed form
  • Polynomial Regression, feature engineering
  • Ridge (L2) & Lasso (L1) regularization
  • Evaluation — MSE, RMSE, R², MAE
Resources
🎓 Andrew Ng ML Specialization — Coursera (START HERE) 📺 StatQuest — Linear Regression (YouTube)
🏷️ Supervised Learning — Classification ▼
  • Logistic Regression — sigmoid, cross-entropy loss
  • Decision Trees — splitting criteria, depth, pruning
  • Random Forest — bagging, feature importance
  • Gradient Boosting — XGBoost, LightGBM, CatBoost
  • Support Vector Machines — kernel trick, margins
  • K-Nearest Neighbors
  • Evaluation — confusion matrix, precision, recall, F1, ROC-AUC
Resources
🎓 Andrew Ng ML Specialization (continued) 📺 StatQuest — Trees, Boosting series (YouTube) 📘 Hands-On ML — Aurélien Géron (must-read)
🔍 Unsupervised Learning ▼
  • K-Means clustering — choosing K, inertia, elbow method
  • DBSCAN — density-based clustering
  • Hierarchical clustering
  • PCA — dimensionality reduction, explained variance
  • t-SNE & UMAP — visualization of high-dim data
  • Anomaly detection basics
Resources
📺 StatQuest — PCA, Clustering (YouTube) ⚡ Kaggle datasets — customer segmentation
🔧 ML Engineering Essentials ▼
  • Train / Validation / Test splits — no data leakage!
  • Cross-validation — k-fold, stratified k-fold
  • Hyperparameter tuning — GridSearch, RandomSearch, Optuna
  • Feature engineering, scaling, encoding categorical vars
  • Handling imbalanced datasets — SMOTE, class weights
  • Pipelines with scikit-learn
Resources
📘 Hands-On ML — Géron (chapters 2–3) 🔧 Scikit-learn docs + user guide (excellent)
⚡ Phase 2 Projects
  • EASY House price prediction (regression end-to-end)
  • EASY Crop yield / disease classification using tabular data
  • MID Customer churn prediction with full ML pipeline
  • MID Kaggle competition — Titanic or House Prices

PHASE 03 ⏱ 8–10 Weeks
Deep Learning
Neural networks, the technology behind modern AI. You'll build systems that see, read, and predict like never before.
🧠 Neural Network Fundamentals ▼
  • Perceptron, activation functions — ReLU, sigmoid, tanh, softmax
  • Forward pass, backpropagation from scratch
  • Loss functions — cross-entropy, MSE, Huber
  • Optimizers — SGD, Adam, AdaGrad
  • Batch normalization, dropout, regularization
  • Learning rate schedules
Resources
🎓 fast.ai — Practical Deep Learning (free, top-down) 🎓 Deep Learning Specialization — Andrew Ng (Coursera) 📺 Andrej Karpathy — Neural Nets: Zero to Hero (YouTube) 🔧 PyTorch (learn this, not TensorFlow first)
👁️ Computer Vision (CNNs) ▼

  • CNN architectures — LeNet, VGG, ResNet, EfficientNet
  • Transfer learning & fine-tuning (biggest practical skill)
  • Object detection — YOLO, Faster R-CNN basics

  • HuggingFace 🤗 — fine-tuning pre-trained models
  • Prompt engineering basics
Resources
📺 Andrej Karpathy — GPT from scratch (YouTube, legendary) 🎓 HuggingFace NLP Course (free, practical) 📘 Natural Language Processing with Transformers — Tunstall et al.

PHASE 05 ⏱ Ongoing
Real World ML & Deployment
This is where most tutorials stop — and where real engineers begin. Building a model is 20% of the job. The rest is making it work in the real world.
🚀 MLOps & Model Deployment ▼
  • FastAPI / Flask — wrap your model as a REST API
  • Docker — containerize your ML app
  • Cloud deployment — AWS/GCP/Azure basics (use free tiers)
  • Model monitoring — data drift, concept drift
  • MLflow — experiment tracking, model registry
  • CI/CD for ML pipelines
Resources
📺 Made With ML (madewithml.com — free MLOps course) 🎓 Full Stack Deep Learning (free) 🔧 Streamlit — fastest way to demo ML apps
🔬 Explainability & Responsible ML ▼
  • SHAP values — why did the model predict X?
  • LIME — local interpretable explanations
  • Bias detection and fairness metrics
  • Model cards — communicating model behavior
Resources
📘 Interpretable ML — Christoph Molnar (free online) 🔧 SHAP library (GitHub)

🎯 Golden Rule

For any new problem: Define the objective clearly → Understand the data → Build a simple baseline → Iterate → Deploy → Monitor. This framework works for 100% of ML problems. Mastery is running this loop faster and with better judgment each time.
